# Release Notes

## Overview

This document summarizes notable changes per release of the **AntiqueBookstore** project.

## Versioning format

Version format `MAJOR.MINOR.PATCH`.

Where:
- MAJOR: breaking changes or major redesign
- MINOR: new features, significant improvements
- PATCH: fixes, small improvements, documentation-only updates

## Release format

Each release entry includes:
- Version / date
- New Features
- Fixes
- Known limitations
- Known issues
- Documentation updates
