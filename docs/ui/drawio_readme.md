# draw.io (diagrams.net)

### How to View and Edit Wireframes in draw.io

The `.drawio` diagram files contain wireframes for the application UI.
They can be opened and edited using the draw.io (diagrams.net) editor.

To work with these wireframes:

1. **Open draw.io:**  Go to the online editor at [https://app.diagrams.net](https://app.diagrams.net) or use the desktop version if installed.
2. **Load the Diagram File:** Choose **Open from** and **Device** and select the `.drawio` file.
3. **Edit the Wireframe:** Use basic shapes (rectangles, text, lines) to represent navigation bars, forms, buttons, tables, and dialogs.
4. **Save the Diagram:** Use **File → Save** to update the `.drawio` file in place.
5. **Export the Wireframe as an Image:** Use **File → Export As → PNG**

For more information on draw.io features, refer to the official site: [https://www.drawio.com/](https://www.drawio.com/).
